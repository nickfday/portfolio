Sep 11: Always a Chance

London, England

I was approached to build the website for Always a Chance, a new charity that aims to prevent violent crime amongst the nation’s youth. I also built a site for their 92 Club Relay fundraiser which saw thousands of people taking part in a relay to all 92 football clubs in England.

The sites were built using wordpress with a blank theme I created. We chose Wordpress as blogging was key to both sites and the vast library of plugins saved a lot of development time. The use of advanced custom fields allowed flexible content that could be managed by non-techy’s using the intuitive administrator view.

I do not see myself as an expert backend developer however creating two dynamic sites from scratch did require considerable PHP coding and MySQL management. In addition to this there are demonstrations of the Google Maps API, social media integration and plugin development.
